registration_tools.create_dataset(data, format, numbers=None, scale=None)[source]¶

Initializes the Dataset object with a list of paths with regex patterns, a list of numbers, a format for the numbers, and a scale for the spatial dimensions.

Parameters:

  • data (list) – A list of paths with regex patterns.

  • numbers (list) – A list of numbers to be substituted in the regex paths.

  • format (str) – The format to be used for the numbers. It can only contain ‘C’, ‘T’, ‘X’, ‘Y’, ‘Z’, each representing a dimension in a multidimensional image. Each character must appear only once.

  • scale (tuple) – A tuple representing the scale for the spatial dimensions. Must be the same length as ndim_spatial.
